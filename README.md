# Alpha Source Site

Static marketing site for `alphasource.com.mx`.

## Deployment

This repository deploys to GitHub Pages through `.github/workflows/deploy-pages.yml`.

## Custom Domain

The repository includes a `CNAME` file for `alphasource.com.mx`.

For the domain to resolve to GitHub Pages, the DNS records for `alphasource.com.mx` must point to GitHub Pages instead of the current GoDaddy/cPanel hosting target.
